# Photo Editor using OpenCV and Streamlit

## Description
A web-based image editing app built using Streamlit and OpenCV.

## Features
- Upload image
- Resize
- Brightness & contrast adjustment
- Filters (grayscale, blur, warm, sharpen)
- Portrait blur
- Download image

## Tools Used
- Python
- OpenCV
- Streamlit
- NumPy

## Run the Project
streamlit run app.py
